# Just One Job Apps

A landing page for Just One Job Apps - a collection of fast, privacy-first micro apps that do one thing extremely well.

## Features

- **Privacy-first**: No ads, no tracking, no analytics
- **No subscriptions**: Free to use
- **Fast & lightweight**: Minimal, focused design
- **Mobile-friendly**: Responsive design that works on all devices

## Apps

### Tip Calculator
A simple tip calculator that helps you figure out tips and splits quickly. Available at [tip.justonejobapps.com](https://tip.justonejobapps.com).

## Development

This is a static HTML site with no build process required. Simply open `index.html` in a browser or serve it with any static file server.
